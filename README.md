# 🛡️ NSLYPAGE — The Official Landing Site

**"Seguridad grado militar, estética hacker, corazón Open Source."**

---

## 📂 Estructura del Proyecto
* **nslypage/** — Carpeta raíz del sitio web.
* **index.html** — Esqueleto semántico y optimización SEO.
* **styles.css** — UI/UX con temática Dark-Terminal y Glassmorphism.
* **script.js** — Lógica para efectos de scroll, ripples y animaciones.
* **LICENSE** — Licencia GNU GPL v2 (Copyleft).
* **README.md** — Guía de despliegue y documentación técnica.
* **firebase.json** — Configuración de Firebase Hosting.
* **.firebaserc** — Identificador del proyecto en Google Cloud.

---

## ✨ Características Principales
* **Diseño Responsive**: Adaptación total a móviles, tablets y escritorio.
* **Estética TUI**: Paleta de colores basada en terminales retro-futuristas.
* **Parallax Dinámico**: Capas de profundidad que reaccionan al movimiento.
* **Interactive UI**: Botones con efecto ripple y barra de progreso de lectura.
* **Optimización**: Carga ultra rápida gracias a la CDN de Firebase.
* **Hecho con Té**: Desarrollado con energía de **muchísimo té**.
