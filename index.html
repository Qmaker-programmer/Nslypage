¿Qué es NSLY VAULT?

Un gestor de contraseñas TUI (Terminal User Interface) que corre directamente en tu terminal. Sin navegador, sin interfaz gráfica bloatware. Solo eficiencia pura y seguridad militar.

$ go run src/main.go
🛡️ NSLY VAULT — ÁREA RESTRINGIDA

Contraseña maestra:
●●●●●●●●

enter entrar • esc salir

🔐 bcrypt cost=14

Verificación de contraseña maestra

🔒 AES-256-GCM

Cifrado de tus contraseñas

🛡️ scrypt (N=32768)

Derivación de clave anti-brute-force

Seguridad de Nivel Militar

Tu bóveda está protegida por las tecnologías criptográficas más avanzadas. Cada contraseña se cifra con AES-256-GCM. La contraseña maestra se verifica con bcrypt cost=14 (tarda ~1 segundo intencionalmente).

Incluso si alguien hackea tu servidor, solo obtiene basura cifrada. 🎯

¿Por qué NSLY?

Si pasas todo el día en la terminal (como todo programador que se respete), NSLY es tu gestor.

No hay bloat. No hay interfaz gráfica lenta. Solo binario estático compilado en Go.

Eficiencia. Seguridad. Libertad. 🚀

Instalación Rápida

git clone https://github.com/Qmaker-programmer/Nsly
cd Nsly
go run src/main.go

# O compilar:
make build
./bin/nsly

Requisitos

  • Go 1.21+
  • Terminal POSIX compatible
  • Linux, macOS o Windows

✨ ¡Eso es todo! ✨
