// 01 — QUÉ ES

¿Qué es NSLY VAULT?

Un gestor de contraseñas TUI (Terminal User Interface) que corre directamente en tu terminal. Sin navegador, sin interfaz gráfica bloatware.

Solo eficiencia pura y seguridad grado militar.

terminal 
$ go run src/main.go

// 02 — SEGURIDAD

Seguridad de Nivel Militar

Tu bóveda está protegida por las tecnologías criptográficas más avanzadas. Cada contraseña se cifra con AES-256-GCM.

La contraseña maestra se verifica con bcrypt cost=14 — tarda ~1 segundo intencionalmente.

💡 Incluso si hackean tu servidor, solo obtienen basura cifrada. 🎯

// 06 — POR QUÉ NSLY

¿Por qué NSLY?

Si pasas todo el día en la terminal (como todo programador que se respete), NSLY es tu gestor.

No hay bloat. No hay interfaz gráfica lenta. Solo binario estático compilado en Go.

🚀 Eficiencia. Seguridad. Libertad.

// 07 — INSTALACIÓN

Instalación Rápida

# Clonar el repositorio
git clone https://github.com/Qmaker-programmer/Nsly
cd Nsly
go run src/main.go

# O compilar el binario:
make build
./bin/nsly

📋 Requisitos

  • 🐹 Go 1.21+
  • 💻 Terminal POSIX compatible
  • 🐧 Linux, macOS o Windows

✨ ¡Eso es todo! ✨
